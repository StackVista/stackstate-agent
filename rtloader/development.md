# Agent Python3 build
```shell
conda create -n ddpy3 python python=3.8

conda activate ddpy3
pip install invoke distro==1.4.0 awscli
inv deps

inv rtloader.clean && inv rtloader.make --python-runtimes 3
inv rtloader.test
```
